This plugin will allow the user to download BAM, FASTQ, and variant caller data through your browser. It makes allowances for either downloading all of the data on a per-barcode basis or archived together into a single zip or tar.bz2 file. Additionally you may configure this plugin to rename the files you are downloading in a dynamic way using the following items.

  • Run Name
  • Report Name
  • Report Date
  • Chip Type
  • Sequencer Name
  • Sample Name
  • Barcode Name
